# Avijit's Portfolio Website

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Portfolio](https://img.shields.io/badge/Portfolio-Visit%20Website-orange.svg)](https://avijitmondal.com)

A modern, responsive personal portfolio website showcasing professional experience, skills, projects, and education.

## Features

- **Modern Design**: Clean, contemporary UI with glassmorphism effects and smooth animations
- **Responsive Layout**: Fully responsive design that works seamlessly across all devices
- **Yellow/Amber Theme**: Eye-catching gradient color scheme with yellow and orange tones
- **Smooth Scrolling**: Interactive navigation with smooth scroll behavior
- **Dynamic Navbar**: Profile photo appears in navigation bar on scroll
- **Project Showcase**: Highlighted personal projects with special visual treatment
- **Contact Form**: Custom modal for contact messages
- **Bengali Logo**: Cultural touch with Bengali character "অ" in the branding

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with custom properties, flexbox, grid, and glassmorphism
- **JavaScript**: Vanilla JS for interactivity and animations
- **Font Awesome 6.4.0**: Icon library for visual elements
- **SVG**: Scalable vector graphics for logo and favicon

## Setup Instructions

1. **Clone or download** this repository
2. **Add your profile photo**:
   - Add your profile photo as `assets/images/photo.jpg`
   - Recommended size: 400x400px or larger (square aspect ratio)
3. **Open** `index.html` in your web browser
4. **Customize** the content with your own information

## Customization Guide

### Personal Information
Edit `index.html` to update:
- Name and title in the hero section
- About me description
- Work experience timeline
- Skills categories
- Project details
- Education information
- Contact information

### Colors
Modify the CSS variables in `assets/css/styles.css` to change the color scheme:
```css
:root {
    --primary-color: #eab308;    /* Yellow */
    --secondary-color: #d97706;  /* Amber */
    --accent-color: #fbbf24;     /* Light yellow */
}
```

### Logo and Favicon
- `assets/images/logo.svg`: Full logo with Bengali character "অ" and name
- `assets/images/favicon.svg`: Browser tab icon with Bengali character
- `assets/images/banner.svg`: Horizontal banner for social media/headers
- All use the same gradient color scheme
